# NPM Packages Documentation

## Published Packages

### aidefence (American spelling)
Core AIMDS functionality with threat detection and response.

```bash
npm install aidefence
```

### aimds
Advanced AI manipulation detection with multimodal and neuro-symbolic capabilities.

```bash
npm install aimds
```

### @midstreamer/wasm
WASM-optimized temporal stream processing with real-time anomaly detection.

```bash
npm install @midstreamer/wasm
```
